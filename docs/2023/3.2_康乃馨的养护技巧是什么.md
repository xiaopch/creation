# 康乃馨的养护技巧是什么

1、土壤

　　康乃馨是须根系草本花卉，最好用比较疏松，土质比较软且透气性比较好的土栽种，因此使用砂质壤土养殖效果最好，但是不要用黏土或者碱性土壤，会让植株营养不良。

　　2、浇水

　　植株是比较耐旱的，除了生长旺季外可适当增加浇水频率和浇水量，其余时段浇水不可过多，保持土壤较湿润即可，一般等待盆土表面见干后再浇透水。

　　3、温度

　　康乃馨的养殖温度不可过高，20℃左右的温度环境最适宜生长。昼夜温差也不可过大，保持在10℃以内。夏季要做好防暑降温工作，冬季则移入室内防寒保暖。

　　4、光照

　　康乃馨对光线要求非常高，属于长日照植物，对于太阳的直射有很强耐受力，平日可直接采取全日照的方式接受光照促进花芽分化延长花期时间。夏季要对植株稍微遮一下光，以防光照太强烈灼伤植株。

　　5、施肥

　　新苗入植前使用厩肥、堆肥、骨粉等有机肥作为基肥，生长旺季薄肥勤施，每7～10天施肥一次，肥料可用腐熟肥或复合肥，但一定要先稀释后再施加。

## 康乃馨养殖注意事项

　　1、虫害：植株比较容易招虫，所以要在平时注意防虫，可以每隔一段时间在花的土壤中喷一些兑水的药剂，来预防虫害的影响。它是一种比较容易生虫的花儿。

　　2、通风：要把植株放在比较通风的环境，这样会有利于它的生长，枝叶也会长得比较茂盛。

## 康乃馨开花后怎么处理

　　康乃馨花谢后，不用太过修剪，以免重剪后影响康乃馨后期的生长开花。康乃馨的花期长，我们只需等植株花朵开败干枯后，用手将干枯的花朵揪下来，或者从康乃馨残花下第一个节位掰断即可。这样清理掉残花的康乃馨，放在光照充足的地方多晒太阳，适当补充肥水，不出一个月，在植株的枝头就又能冒出花苞了。
