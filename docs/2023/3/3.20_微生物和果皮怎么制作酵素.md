## 微生物和果皮怎么制作酵素
```
微生物制作酵素的方法一般包括以下步骤：

1. 培养微生物：选择适合制作酵素的微生物，如曲霉、酵母、乳酸菌等，并在特定的培养条件下进行培养，如保温、搅拌、通风等条件。

2. 加入原料：将需要发酵的原料加入培养基中，如蔬菜、水果、粮食等。

3. 发酵过程：控制培养条件，使微生物进行发酵。发酵过程中产生的相关物质便是酵素。

4. 提取酵素：发酵完成后，将发酵液进行离心或过滤等处理，得到目标酵素。

果皮制作酵素的方法如下：

1. 准备原料：将需要用来制作酵素的果皮进行清洗、去污、切碎等处理，以便后续的发酵。

2. 热处理：将果皮进行一定的热处理，如煮沸或微波处理，以去除细菌和其他微生物，保证发酵的纯度。

3. 加入酵素发酵剂：将果皮放入容器中，加入酵素发酵剂，如酵母、面包酵母、酸奶等。

4. 发酵：将容器密封，并放在一定温度下进行发酵。发酵过程中产生的相关物质便是酵素。

5. 筛选提取：发酵完成后，将发酵液通过筛网滤取，得到目标酵素。
```
