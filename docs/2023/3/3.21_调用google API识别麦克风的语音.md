## 调用google API识别麦克风的语音



```
##从终端安装 SpeechRecognition
##使用命令：pip3 install SpeechRecognition
##pip install pyaudio
import speech_recognition as sr
print(sr.__version__)

import os
os.environ["HTTP_PROXY"] = "http://127.0.0.1:7890"
os.environ["HTTPS_PROXY"] = "http://127.0.0.1:7890"

r = sr.Recognizer()

mic_list = sr.Microphone.list_microphone_names()
##列举系统中所有的麦克风设备
##print(mic_list)

##开始录音
mic = sr.Microphone(device_index=1)
print('录音中...')
with mic as source:
    r.adjust_for_ambient_noise(source,duration=0.1)
    #参数: source, timeout = None,phrase_time_limit = None snowboy_configuration =None
    audio = r.listen(source,15)
print('录音结束...')

##将录音保存为文件
wav_num = 1
with open(f"00{wav_num}.wav", "wb") as f:
            #将麦克风录到的声音保存为wav文件
            f.write(audio.get_wav_data(convert_rate=16000))

##调用google API进行语音识别
result = r.recognize_google(audio, language='zh-CN', show_all= True)
print(result)


##SpeechRecognition 的核心就是识别器类。
##
##Recognizer API 主要目是识别语音，每个 API 都有多种设置和功能来识别音频源的语音，分别是：
##
##recognize_bing(): Microsoft Bing Speech
##
##recognize_google(): Google Web Speech API
##
##recognize_google_cloud(): Google Cloud Speech - requires installation of the google-cloud-speech package
##
##recognize_houndify(): Houndify by SoundHound
##
##recognize_ibm(): IBM Speech to Text
##
##recognize_sphinx(): CMU Sphinx - requires installing PocketSphinx
##
##recognize_wit(): Wit.ai
##————————————————

```
