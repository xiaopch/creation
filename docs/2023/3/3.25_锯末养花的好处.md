## 锯末养花的好处

- 生根快，开花早，不烂根！

  我们知道锯末可以用来发蘑菇，而我们买的很多便宜的通用营养土，也都是菌种废料做成的。说到这您应该想到了，其实锯末是可以作为栽培土养花的。锯末在园艺养花中主要有以下5个方面的用途。

**锯木在家庭养花中的运用**

**1、扦插基质**

  养花很多人都喜欢扦插繁殖，而锯末是非常好的扦插基质，可以用来扦插月季、栀子花、石榴等木本花卉，由于锯末具有非常好的透气性，所以用于扦插生根非常的快。

**2、高空压条基质**

  除了扦插，高空压条时使用锯末和泥土各一半作为包裹材料，可减轻枝条的重量负担，同时又增加了泥炭的透气性，有利于生根。

**3、发酵做肥土**

锯末也是非常好的腐肥材料，可将锯末与有机肥混合发酵，发酵的时间因温度会有不同，夏季约经过40天左右的发酵，就可以使用了。使用时应与其他的基质混合使用，以免肥力过高造成烧苗。

**4、缓苗**

  现在我们都喜欢网购花苗，很多花苗经过长途的邮寄，到手上时可能已经严重失水了，这时如果栽入含肥量较高的营养土中，花苗很容易被烧死，而用没有肥力的土栽培时，又容易因突然板结不透气，而把苗闷死。这时可以用锯末来种植新苗，锯末可以起到透气，保湿的效果，等新苗长出新根后，再重新移栽，可大大降低苗的死亡率。

**5、可代替兰花泥使用**

  喜欢养兰花的朋友都知道，兰花需要专用的土壤栽培。其实锯末是非常好的养兰花的植料。锯末可完全替代兰花泥，用锯末种植兰花，可以让兰花的假鳞茎更大，优雅肥壮，根呈半透明凝脂状，不烂根。而且还可以让兰花的开花时间提早。

  锯末在盆内会慢慢腐烂，可给兰花提供足够的养分，同时锯末适合菌类生长，可使兰花共生的兰菌生长良好，从而使兰根长期处于最佳的状态。
