#### 文字转语音TTS

```
from paddlespeech.cli.tts import TTSExecutor
tts_executor = TTSExecutor()
wav_file = tts_executor(
    text="人类终究要走向宇宙,暗能量是什么，平行宇宙存在吗",
    output='audio.wav',
    am='fastspeech2_csmsc',
    voc='mb_melgan_csmsc',
    lang='zh',
    spk_id=0)
```



#### 语音识别ASR

```
from paddlespeech.cli.asr.infer import ASRExecutor
audio_file = "audio.wav"

# 转换音频的采样频率
import os
import librosa
import soundfile

new_sample_rate = 16000
signal, sr = librosa.load(audio_file, sr=None)      #调用librosa载入音频
new_signal = librosa.resample(signal, sr, new_sample_rate)      #调用librosa进行音频采样率转换
soundfile.write(audio_file, new_signal , new_sample_rate)

asr = ASRExecutor()
result = asr(audio_file=audio_file, model='conformer_online_wenetspeech')
print(result)
```



#### ffmpeg转换音频的采样率

```
ffmpeg -y -i input.wav  -ac 1 -ar 16000  -b:a 16k  output.wav
```

#### from timer import timer 问题的解决办法

- 删除旧的timer.pyd文件

```
del C:\Users\******\AppData\Roaming\Python\Python37\site-packages\win32\timer.pyd
```

- 安装新的timer库

```
pip install codetiming 
## from codetiming import Timer
```
