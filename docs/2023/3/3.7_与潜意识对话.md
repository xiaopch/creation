### 3.7_与潜意识对话

```
大家好,今天我们一起用跟随我5分钟的时间
通过催眠和和自己的潜意识对话
告别自己所有的委屈和焦虑
重新开始

在开始今天的催眠体验之前,有三个注意事项:
一,今天的催眠,会让你有一次,自我原谅的体验,你需要对我绝对的信任。
二,你需要绝对的放松,尽情地释放你的情绪。
三,你需要找一个,让你感觉安全,又舒适的环境。

现在正式开始我们的催眠体验。
请你根据我的引导,慢慢地闭上眼睛。
跟我一起做两个深呼吸。

关注自己的呼吸,你会感觉到身体越来越放松。
当我数到五,你的全身都会处于最放松,最舒服的状态。
一,你的听觉系统开始放松下来,感觉所有的声音都非常的美妙
都在按摩着你的头脑,使你更加的放松。
二,你的脸部放松了,嘴角上露出了淡淡微笑
三,你的你的腹部和胸腔放松了
四,你的四肢放松了,你的手指和脚趾也慢慢放松了
五,你的身体也越来越放松,越来越放松。

现在你的每一根神经都处于最放松,最舒服的状态。
你的呼吸也变得越来越均匀,越来越平和。
请你再跟着我做两个深呼吸。

当我数到三,你的面前会出现另外一个自己。
这一个自己包含了你所有的委屈,所有的压力,所有的焦虑。
你不知道它叫什么名字,也不知道它长什么样子,让我们一起来看一看。
一,二,三。
我会给你30秒的时间,你只需要做一个动作,说七个字。
请你紧紧地抱住另外一个你,并且对他说,对不起,我原谅你。
......30S......
当我数到三,你面前另外一个自己会消失不见,同时带走你所有的忧虑,恐惧和压力。
一,二,三。
请你做一个深呼吸,调整一下自己的情绪。
我会再数三个数,当我数到三的时候,你会回到现实中来
并且感觉自己的身体非常的舒服,非常的放松。
一,二,三
希望你今天,在原谅自己的催眠体验中,能有收获,你可以收藏本视频,并关注我
我是C,一个普通的催眠疗愈师。
```
