## 花青素含量高的食物有哪些

```
  通常来讲，并没有含花青素最多的食物，但是茄子、紫葡萄、紫薯、黑枸杞、紫甘蓝、黑桑葚、蓝莓、杨梅、黑芝麻、血橙、樱桃、草莓、红心火龙果等食物中，含有一定量的花青素。

  花青素又称为花色素，是一种水溶性的植物色素，属于黄酮类化合物。花青素不但含于花中，亦含于根、叶、果皮及种皮等部分中。有研究表明，花青素对体内的粥样硬化斑块有着一定抑制作用，并且对脂质过氧化物有明显的清除作用。花青素对细胞突变还有辅助的抑制作用，并且还具有一定的抗氧化作用，通常存在于水果、花卉等植物中，常见的包括黑枸杞、紫甘蓝、黑桑葚、蓝莓、杨梅、黑芝麻等。适量食用上述食物，可以提高抗氧化能力，有利于机体的健康。

  若大量吃了富含花青素的食物后，导致胃痛、胃胀等现象，建议及时就诊于消化内科，通过完善*胃镜*、*血常规*、便常规等检查明确诊断，再进行相应的治疗，在日常生活中需要注意适量补充，不要补充过多加重胃肠负担。
```
