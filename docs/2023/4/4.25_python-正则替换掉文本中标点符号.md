## python-正则替换掉文本中标点符号

```
#!/usr/bin/python3
# coding: utf-8
import re

# 正则去掉所有的标点符号
def drop_re(text):
    punc = '~`!#$%^&*()_+-=|\';"＂:/.,?><~·！@#￥%……&*（）——+-=“：’；、。，？》{《}】【\n\]\[ '
    str = re.sub(r"[%s]+" % punc, "", text)
    return str

str="爱编程，一个运维兼程序员的博客！"
print(drop_re(str))
```
