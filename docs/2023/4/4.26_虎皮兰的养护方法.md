# 虎皮兰的养护方法

李哲馨 副教授 植物学博士

1、光照：虎尾兰需要有充足的光线，在光线充足的条件下生长良好，长期处于阴暗环境会导致虎皮兰叶子发黄；

2、土壤：选择排水性能好的沙质土壤；

3、温度：虎皮兰适宜的温度为20~30℃，冬季温度需保持在10℃左右；

4、浇水：夏季3至4天浇水一次，冬季一周一次；

5、肥料：每月施肥两次，需要施稀薄肥液。



　　虎皮兰也叫虎尾兰，它的品种较多，株形和叶色变化较大，主要品种有金边虎尾兰和银脉虎尾兰，适合布置装饰书房、客厅、办公场所，可供较长时间观赏。

　　虎皮兰适应性强，性喜温暖湿润，喜光又耐阴，平时用清水擦洗叶面灰尘，保持叶片清洁光亮。春季根颈处萌发新植株时要适当多浇水，保持盆土湿润；夏季高温季节也应经常保持盆土湿润；秋末后应控制浇水量，盆土保持相对干燥，以增强抗寒力。

　　虎皮兰的管理较为粗放，喜疏松的沙土和腐殖土，耐干旱和瘠薄；盆栽可用肥沃园土3份，煤渣1份，再加入少量豆饼屑或禽粪做基肥。盆栽虎皮兰一般两年换一次盆，以提高植株的生长速度。
