## 怎样在Windows下安装python

> - 每天用心解决一个问题
>
> - 用持续性小赢来创造社会价值
> - 讲故事来做教育
>



1. 打开浏览器软件,常见的浏览器有Internet Explorer,Google Chrome,Microsoft Edge,360安全浏览器等,打开任何一个都可以.

2. 在浏览器地址栏输入python的官方网址: <u>https://www.python.org/</u> ,按下Enter键进入python官方网站

3. 在python官网选择 <u>Downloads</u> 菜单,选择Windows进入python的windows版本下载页面

4. 选择Python 3.7.4的版本下载,网址为: <u>https://www.python.org/downloads/release/python-374/</u>

5. 根据自己windows的版本下载匹配的python版本,我选择的是 [Windows x86-64 executable installer](https://www.python.org/ftp/python/3.7.4/python-3.7.4-amd64.exe) 版本,下载后的文件在浏览器默认的下载文件夹,一般是系统的下载文件夹,文件名是<u>python-3.7.4-amd64.exe</u>.

6. 鼠标双击<u>python-3.7.4-amd64.exe</u>文件开始安装python,安装时勾选安装选项<u>Add python.exe to PATH</u>,将python解释器加入到系统命令行的路径中,以便后面方便运行python脚本程序,完成python程序安装.

7. 同时按下<Win> + <R>键打开Windows 命令运行窗口,输入<u>python</u>, 并点击<确定>打开python解释器.

8. 在python命令行窗口输入: 

   ```
   print('Hello World') 
   # 输出: Hello World 
   # 表示python程序安装成功.
   ```
