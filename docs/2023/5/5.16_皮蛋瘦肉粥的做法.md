## 皮蛋瘦肉粥的做法

#### 食材原料

| 大米150g     | 猪里脊肉100g  |
| ------------ | ------------- |
| **皮蛋2个**  | **清水适量**  |
| **香葱2根**  | **生姜1小块** |
| **食盐适量** | **鸡精适量**  |
| **香油适量** | **生粉适量**  |
|              |               |

#### 制作方法

| 步骤 | 具体操作                                                     |
| ---- | ------------------------------------------------------------ |
| 1    | 将适量的大米洗净沥干，加入香油，搅匀后放置一旁，浸泡15~30分钟； |
| 2    | 香葱洗净切碎，姜去皮切成细丝；                               |
| 3    | 皮蛋剥皮，洗净后，切成小块备用；                             |
| 4    | 锅中先倒入香油浸泡后的米，再倒入清水，最后倒入姜丝，大火煮开后，改成小火煮20分钟，期间每隔5分钟用勺子沿同一方向搅拌一次，以免黏锅底; |
| 5    | 把里脊肉先切成片，再切丝加入食盐，搅匀腌制10分钟左右，撒上生粉（先不要急着拌匀，等下锅前再拌匀最好）； |
| 6    | 当粥煮到米完全熟透，粥水也比较浓稠时，放入皮蛋，继续煮10分钟； |
| 7    | 然后把腌制好的里脊肉丝搅拌均匀后，倒入粥中；                 |
| 8    | 调入食盐·鸡精后，并用勺子不断搅动；                          |
| 9    | 倒入香油搅拌；撒上香葱碎即可盛出食用。                       |
| 10   | 质地黏稠、口感顺滑、味道独特的皮蛋瘦肉粥就做好了。           |
