## 百度大模型Prompt设计课程

#### Prompt万用公式

Prompt = 任务 +生成主体 +细节(可选)+形式(可选)

###### [必选]

**任务**:指明希望模型完成的任务类型，如“请生成一篇广告文案”,“扮演翻译官”,"生成一篇广告文案"即是希望生成的任务类型。
**生成主体**:指要生成的主要对象，如“请生成一篇关于狗不理包子的广告文案”中，“狗不理包子”即是主要对象

###### [可选]

**细节**，倘若我们需要让生成文案中增加emoi表情，则可以在Prompt中增加“加一些emoi表情点缀”等细节描述来丰富最终生成效果，但通常细节性描述不一定能够完全生效

**形式**:指明生成后的排版、内容风格，如“用文本描述”“代码形式生成”“转为广东话”等

```
我希望你充当讲故事的人，你要想出一个童话故事。
```

```
我希望你充当讲故事的人，你要想出一个中华文化的童话故事，要求内容积极向上，与猫猫有关。请用广东话讲一下。
```
