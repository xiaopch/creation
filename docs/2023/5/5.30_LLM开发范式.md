#### 微调

###### zero-Shot 

```
Prompt='''
判断给定句子的情感倾向，用一个单词回答: 正向、负向、中性
句子: 我们是快乐的年轻人。
情感:'''
```

###### Few-Shot

```
Prompt='''
判断给定句子的情感倾向，用一个单词回答: 正向、负向、中性。
句子:今天真开心。
情感:正向
##
句子: 心情不太好情感: 负向
##
句子:我们是快乐的年轻人
情感:
'''
```

###### Fine-Tune # 微调

```
Prompt='''
今天真开心。-->正向
心情不太好。-->负向
推理
我们是快乐的年轻人。-->
'''
```
