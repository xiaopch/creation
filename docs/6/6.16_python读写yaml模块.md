## python读写yaml模块

Python 中的 `yaml` 模块提供了一些函数和类，用于读取和写入 YAML 格式的数据。以下是一些常用的操作。

#### yaml文件介绍

YAML是一种简洁的非标记语言。其以数据为中心，使用空白，缩进，分行组织数据，从而使得表示更加简洁。

1. yaml文件规则
   基本规则：
       大小写敏感
       使用缩进表示层级关系
       缩进时不允许使用Tab键，只允许使用空格。
       缩进的空格数目不重要，只要相同层级的元素左侧对齐即可
       使用#表示注释
       字符串可以不用引号标注

2. yaml文件数据结构

​    对象：键值对的集合（简称 "映射或字典"）
​    键值对用冒号 “:” 结构表示，冒号与值之间需用空格分隔
​    数组：一组按序排列的值（简称 "序列或列表"）
​    数组前加有 “-” 符号，符号与值之间需用空格分隔
​    纯量(scalars)：单个的、不可再分的值（如：字符串、bool值、整数、浮点数、时间、日期、null等）
​    None值可用null可 ~ 表示  



#### 写入 YAML 文件

使用 `yaml.dump()` 函数可以将 Python 对象转换为 YAML 格式的数据，并将其写入文件中。例如：

```python
import yaml  
  
data = {'name': 'xiaop', 'age': 25}  
  
with open('data.yaml', 'w') as f:  
    yaml.dump(data, f)
```



#### 读取 YAML 文件

使用 `yaml.safe_load()` 函数可以从 YAML 文件中读取数据，并将其转换为 Python 对象。例如：

```python
import yaml  
  
with open('data.yaml', 'r') as f:  
    data = yaml.safe_load(f)
    
print(data)
```



#### 读取和写入 YAML 流

使用 `yaml.load()` 和 `yaml.dump()` 函数可以从 YAML 流中读取和写入数据。例如：

```python
import yaml  
  
# 从 YAML 流中读取数据  
stream = yaml.dump([1, 2, {'a': 'b'}])  
data = yaml.load(stream)  
  
# 将数据写入 YAML 流中  
stream = yaml.dump(data)
```

#### 使用 YAML 的注释功能

YAML 支持注释功能，可以使用 `yaml.comments.CommentedMap` 类来创建带有注释的 Python 对象。例如：

```python
import yaml  
from yaml.comments import CommentedMap  
  
data = CommentedMap()  
data['name'] = 'Alice'  # 注释1  
data['age'] = 25  # 注释2  
data['hobbies'] = ['reading', 'swimming']  # 注释3  
data['#'] = '这是注释行'  # 注释4  
data.yaml_set_start_comment('这是一个示例数据\n', 1)  # 注释5  
data.yaml_set_end_comment('end of data', -1)  # 注释6  
print(yaml.dump(data))  # 输出带有注释的 YAML 数据
```
