## Microsoft Visual C++ 14.0 or greater is required问题解决方案

- python模块安装时出现Microsoft Visual C++ 14.0 or greater is required问题解决方案

#### 测试python版本

Python 3.9.12



#### 删除已安装的VS和VC版本



#### 升级pip

```
python.exe -m pip install --upgrade pip
```



#### 下载Microsoft C++ 生成工具离线版本ISO文件

```
2021/04/27  06:30           　 VCPlusPlusBuildTools2015Update3_x64_Files.cat
2016/06/21  06:57        　　 VisualCppBuildTools_Full.exe
2021/04/27  06:24    <DIR>          packages
```

- 用虚拟光驱打开ISO文件点击VisualCppBuildTools_Full.exe默认安装就可以了

  

#### 安装webrtcvad模块

```
pip install webrtcvad
```
