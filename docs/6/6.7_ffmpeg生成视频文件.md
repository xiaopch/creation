## ffmpeg生成视频文件

####  



#### 直接无损合并

```
ffmpeg.exe -f concat -safe 0 -i video_files.txt -c copy -y video_out.mp4

-f concat是指合并

-safe 为了避免权限报错

-c copy 一定要指定，不然会重新封装，数据文件就会变大。

好处：文件不会变大，且加了 -c copy很快完成合并。
```

video_files.txt内容示例如下:

```
file '1.png.mp4'

file '2.png.mp4'

file '3.png.mp4'

file '4.png.mp4'
```



#### 图片和音频合成视频

```
ffmpeg -loop 1 -framerate 1 -i 2.jpg -i audio.wav -map 0:v -map 1:a -r 10 -movflags +faststart -shortest -fflags +shortest -max_interleave_delta 100M out.mp4
```

```
ffmpeg -framerate 1 -f image2 -loop 1 -y  -i 2.jpg -i audio.wav -movflags +faststart -shortest -fflags +shortest   -r 25  output.mp4
```



```
ffmpeg -framerate 1 -f image2 -loop 1 -y  -i 2.jpg -i audio.wav -movflags +faststart -shortest -fflags +shortest  -s 1080*1920 -r 25  output.mp4
```

-framerate 速率，越小每张图片停留时间越长
-loop 循环一遍文件夹内的图片
-i 图片路径，正则匹配
-i 音频地址
-y 覆盖已有生成文件
-s 画质
-r 帧数
-t 总时长
