# 生命就是无数个爱自己的机会——露易丝·海

生命就是无数个爱自己的机会

过去，现在，未来

爱自己，每个部分的自己

在我心深处，

有个无限的爱的泉源。

我现在允许让这份爱浮出表面——

它充满了我的心、我的身体、

我的心智、我的意识、我的存在，

从我身上向四面八方散发出去，

并且加倍地回到我自己。

我越是运用与给出爱，

我越是需要去给予——

供给是源源不绝的。

运用爱让我感觉很好。

这是我内在喜悦的表达。

我爱我自己，

因此我充满爱地照顾我的身体。

我带着爱以营养的食物与饮料滋养它。

我带着爱照料它并且装扮它，

而我的身体也以鲜活的健康与能量，

充满爱地回应我。

我爱我自己，

因此我给自己一个舒适的家，

一个可以满足我所有的需求，

并且令人愉快的地方。

我以爱的振动频率填满房间，

这样所有进来的人，

包括我自己，

都会感受到这份爱，

并且被它滋养。

我爱我自己，

因此我做我真正享受喜爱的工作，

这个工作使用了我的创造天赋与能力，

和我所喜爱并喜爱我的人一起工作，

并且赚取很好的收入。

我爱我自己，

因此我以爱的方式对待所有的人，

不论是行为或是思考——

因为我知道凡是我所给出的，

将会加倍回到我的自身。

在我的世界里我只会吸引充满爱的人，

因为他们就是我所是的镜子。

我爱我自己，

因此我宽恕必且完全地释放过往的经验，

我是自由的。

我爱我自己，

因此我全然地活在当下，

经验每个时刻皆美好，

并且知道我的未来是光明、喜悦与安全的——

因为我是宇宙挚爱的孩子，

宇宙充满着爱地照顾着我，

现在到未来。

心诚所愿。

 作者：露易丝·海
