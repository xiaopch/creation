# 两种自制生根粉的方法

只要懂些养花知识的人都知道，大部分常见的花卉都是靠扦插的方法繁殖。扦插繁殖的方法虽然简单，但往往会出现扦插枝条不生根的情况。因此网上很多教程会推荐花友们在扦插过程中使用生根粉。

生根粉可以从花市购买到。如果不想购买的话，我们也可以利用身边常见的“小东西”进行配置。下面就为大家介绍2种生根粉的自制方法：

## **1.红糖+维生素B自制生根粉**

**红糖的生根原理**

红糖是大家厨房里常备的材料。红糖中含有葡萄糖、果糖等多种单糖和多糖类物质，适当的浓度下，可以为植物提供生长需要的营养，增加植物细胞的活力。此外，红糖中还含有叶酸、微量元素、氨基酸、赖氨酸等物质，能够加快细胞代谢，诱导植物枝条更快生根。在有机农业上，也常常会用红糖和酵母的发酵液来帮助蔬菜的根系生长。

**维生素b族片生根原理**

维生素b族片，是很多家庭常备的一种维生素片。维生素B族片中包括维生素b1、b2、b3、b6到b12，都是水溶性维生素。其中维生素b2这种成分，溶于水之后会分解出水杨酸。水杨酸具有抗菌作用，可以防止枝条的伤口腐烂，以及促进根系的发育。一般家庭常备的是维生素b族片，如果花友们能用纯维生素b2药片，效果会更好。

**制作方法**

用红糖和维生素b族片自制生根粉的方法非常简单：取1片维生素b片和30克红糖，将维生素b片用工具碾成粉末，粉末越细越好。然后取500~800毫升的清水，将药片粉末和红糖放入清水中，注意所用的清水不能是热水，否则会破坏药片的成分。

将水搅拌，保证红糖和药片粉末充分溶解在水里。在扦插之前，将插穗枝条浸泡在溶液中15~20分钟，让枝条吸收药液成分，随后再正常进行扦插，用此方法，枝条的生根率会大大提升。

通常情况下，每次扦插完之后都会剩下不少生根液。花友们可以按照1:5的比例，将生根液中加入大量的清水，再次搅拌均匀。然后用这些溶液浇灌花卉，能促进盆栽花卉的根系发育，让花草长得更加茂盛。

## **2.白糖自制生根粉**

花友们家里若是没有红糖和维生素b族片，也可以用白糖来自制生根粉。白糖能促进枝条生根的原理与红糖类似，不过白糖中只含有糖类物质，不含氨基酸、叶酸和微量元素，也没有维生素b片中类似水杨酸的成分。所以它的生根效果会比红糖加维生素B片的效果差一些，但也能起到不错的促进生根的效果。

白糖生根水的制作更加简单，只要事先准备少量白糖和凉白开水就可以了。白糖可以用绵白糖或者白砂糖，水则必须是凉白开或者纯净水，不能用自来水。把白糖和水，按照1:60的比例混合，搅拌均匀制成糖水。在扦插前，取1克的白糖兑入500毫升的清水，制成0.2%的白糖水溶液。将扦插用的枝条根部在白糖水溶液中浸泡30分钟，取出晾干。随后把晾干的枝条用于正常扦插即可。
